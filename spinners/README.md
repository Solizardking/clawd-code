Install themed spinner verb packs for Clawd agents directly from the solana-clawd repository.
Powered by [x402.wtf](https://x402.wtf) · $CLAWD: `8cHzQHUS2s2h8TzCmfqPKYiM4dSt4roa3n7MyRLApump`

## Installation

```bash
npx skills add Solizardking/solana-clawd
```

## Usage

Once installed, ask your Clawd agent:

```
Install a spinner pack
```

You'll be presented with 45+ themed packs to choose from, including:

- **Developer** — Programming & dev culture
- **Pirate** — Pirate speak
- **Vibecoder** — Vibe coding culture
- **Corporate** — Corporate buzzwords & jargon
- **Space** — Space & sci-fi
- **Wizard** — Fantasy & magic themed
- **Vim** — Vim editor enthusiasts
- **Darth Vader** — Star Wars' Sith Lord
- **Yoda** — Star Wars' Jedi Master
- **Gordon Ramsay** — Angry chef yelling at code
- **Jack Sparrow** — Chaotic pirate captain
- And many more!

The selected pack will be installed into your `~/.clawd/settings.json` (or `~/.claude/settings.json` as fallback).
