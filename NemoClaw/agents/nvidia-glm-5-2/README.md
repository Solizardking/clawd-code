# NVIDIA GLM 5.2 OpenShell Agent

This agent keeps Nemo Clawd's default inference path on Z.AI GLM-5.2 through
`ZAI_API_KEY`. OpenRouter is only an assist route for NVIDIA Nemotron fallbacks
and provider comparison.

The agent wallet is private to OpenShell:

- Scope: `agent-private`
- Name: `nvidia-glm-5-2`
- State path: `~/.nemoclawd/openshell-wallets/nvidia-glm-5-2`
- Private key export: disabled
- Signing: requires explicit human approval

No key material is stored in this repository.
